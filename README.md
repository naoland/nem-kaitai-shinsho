# NEM解体新書

## はじめに

本ドキュメントは、[nemlog](https://nemlog.nem.social/)で[目指せ北海道](https://nemlog.nem.social/profile/416)氏が作成した**NEM(NIS1)のホワイトペーパーである[NEM Technical Reference](https://nemplatform.com/wp-content/uploads/2020/05/NEM_techRef.pdf)の解説付き和訳記事**を[YUTO](https://nemlog.nem.social/profile/2873)氏が目次としてまとめてくださった内容を、[naoland](https://github.com/naoland)がマークダウン記法で整理したものです。
それぞれの記事についてもマークダウンとして整理しました。

nemlogに投稿されているYUTO氏の目次記事は[こちら](https://nemlog.nem.social/blog/44109)をご覧ください。

このドキュメントは現在整理中ですが、完了後には一括でダウンロードしていただけるようになります。
すでに公開されているPDFも（許可が得られれば）同梱したいと思います。
